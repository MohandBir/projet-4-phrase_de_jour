# projet 4 : Phrase du jour

## Description du projet : 
    Phrase du jour :
        -Application web développée avec Symfony permettant d’afficher, liker et commenter des « phrases du jour ».
        -Le projet met en œuvre une architecture complète avec Doctrine ORM, des templates Twig et une gestion du CRUD.
        -Un système d’authentification permet aux utilisateurs de s’inscrire, se connecter et interagir avec les contenus.
        -Un espace administrateur sécurisé offre la gestion des phrases (ajout, modification, suppression).
        L’interface utilise Bootstrap (Bootswatch Brite) pour un rendu moderne et responsive.
